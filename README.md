# VideoDistortion
Very slow video distorter

## Requirements 
* python3
* ffmpeg
* ImageMagick
* opencv

## How it works
Parses the video into frames and distort each of them

## How to use:
```
$ sudo apt install -y ffmpeg python3 imagemagick
$ pip install opencv-python
$ git clone https://github.com/m4xx1m/VideoDistortion
$ cd VideoDistortion
$ python3 main.py
```
