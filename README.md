# VideoDistortion
very slow video distorter

## Requirements 
* python3
* ffmpeg
* ImageMagick

## How it works
Parses the video into frames and distort each of them

## How to use:
```
$ sudo apt unstall -y ffmpeg python3 imagemagick
$ git clone https://github.com/m4xx1m/VideoDistortion
$ cd VideoDistortion
$ python3 main.py
```
